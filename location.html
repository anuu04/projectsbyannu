Location Management Flow

The goal was to simplify how a photography company manages multiple studio locations. The existing process required manual linking of data and lacked clear visibility between the admin and client sides. The redesigned flow ensures seamless coordination between admin and client platforms, maintaining data accuracy across both. It focuses on intuitive navigation, consistent information hierarchy, and a clean interface that scales as the number of locations grows.

Project Overview

  • Problem Statement

    Admins struggled to maintain consistent information across locations, and clients couldn’t easily explore detailed studio profiles with up-to-date data.

  • Solution

    I designed a connected Location Management System where

    admins can seamlessly:

    • Add new locations with basic info.

    • Update remaining details on a dedicated profile page.

    • Automatically link reviews, bookings, and staff once associated.

    On the client app, users can:

    • Add new locations with basic info.

    • Update remaining details on a dedicated profile page.

    • Automatically link reviews, bookings, and staff once associated.

  • Impact

    • Reduced admin workload and data duplication.

    • Enhanced client experience with accurate, visually engaging location info.

Responsibilities

  • User Flow & Wireframing

    Mapped out the complete interaction between admin and clients to ensure smooth transitions between adding, editing, and viewing studio locations.

  • Dashboard & UI Design

    Designed an intuitive admin dashboard for managing studios, linking bookings, and updating details with minimal effort.

  • Client App Design

    Crafted a visually appealing client interface to explore nearby studios, view reviews, and navigate directly via integrated maps.

  • System Integration

    Planned how studio data (reviews, staff, bookings) automatically connects with respective locations to maintain real-time accuracy.

  • UI Development

    Converted high-fidelity designs into responsive front-end layouts using HTML, CSS, and JavaScript while ensuring consistency across screens.
